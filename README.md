coursera-scala
==============

Functional Programming Principles in Scala at Coursera


Course Syllabus:
----------------
1. Programming paradigms; overview of functional programming and the Scala programming language.
2. Defining and using functions, recursion and non-termination, working with functions as values, reasoning by reduction. 
3. Defining and using immutable objects, review of inheritance and dynamic binding.
4. Types and Pattern Matching
5. Working with Lists
6. Collections and Combinatorial Search
7. Lazy Evaluation

More info: https://www.coursera.org/course/progfun
